## 旁白

现在的 AI 编程，最累的往往不是写代码。是你要盯着它，催着它，还要记住每个窗口到底做到哪了。

Chill Vibe 把散落的会话变成一张工作台。一个项目一列，一个任务一张卡，多条工作同时推进。

前端在修，后端在查，测试在跑。所有 Agent 都留在原来的位置，你不用在十个终端之间来回找进度。

Claude 不合适，就切到 Codex。上下文跟着走，切回来，原会话还能继续。中断可以恢复，想试另一条路，就从任意消息分叉。

长任务也不用一直守着。它会继续跑，必要时自动追问，完成后主动回来汇报。离开电脑，用手机就能查看、追加指令或者停止任务。

所有命令、文件改动和任务进度，都清楚地留在聊天里。危险删除和破坏性 Git 操作，还有独立防护。

Chill Vibe 不想替代你熟悉的工具。它只做一件事，让多个 AI 真正像一支团队一样工作，而不是一排需要你照看的聊天框。

少盯窗口，多做判断。
