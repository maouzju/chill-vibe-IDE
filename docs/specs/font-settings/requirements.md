# 字体设置需求

## 背景
用户希望在设置里能选择应用字体，并且默认字体必须排在第一位，保持当前视觉不变。

## 需求
1. 设置 → 外观 增加“字体”下拉选项。
2. 默认选项置顶，字体栈等同当前应用字体：`Aptos`, `IBM Plex Sans`, `Segoe UI Variable`, `Segoe UI`, `system-ui`, `sans-serif`。
3. 选择字体后立即作用到整个应用，并随设置持久化。
4. 老用户没有该字段时自动回落到默认字体。
5. 重置界面默认值时同时恢复默认字体。
6. 内置字体选项必须覆盖常用 Windows 中文字体、英文无衬线、英文衬线和等宽字体。
7. 字体下拉旁或下方必须提供即时预览，让用户不必切回主界面也能判断字体效果。

## 验收
- 新默认设置包含默认字体。
- 非法/缺失字体设置会被规范化为默认字体。
- 设置 UI 可选择默认字体、常用中文字体、英文字体和等宽字体。
- 切换字体时，预览文案和应用主体字体同步变化。
